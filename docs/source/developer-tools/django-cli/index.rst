Django CLI Package
==================

.. title:: Django CLI Package - Open-source helpers for managing Django Internals and more
.. meta::
    :description: Django Helpers for managing the environment, migrations, settings, routing, files, GIT and dataBase backup 
    :keywords: django cli, generate django models, manage environment, manage migrations, manane Django codebase  

Open-source helpers for managing Django internals with ease. This package in included in the following starters: 

- `Django Datta Able </product/datta-able/django/>`__ - Free Product
- `Dynamic Django <https://dynamic-django.onrender.com/>`__ - Paid starter, part of the `Onboarding Kit </onboarding-kit/>`__ Bundle

All helpers provided by the package are listed below. 

Features 
--------

- `Interact with Django Internals <./query-django.html>`__
    - Get Django Instance
    - Check DB Connection
    - List registered Applications 
    - List Models 
    - Print Models fields, foreign keys 
- Manage Environment Variables 
- Manage Dependencies
- Control Project Settings
- Update Routing 
- Update Models
    - Add/Remove Models 
    - Add/Remove/Update Fields         
- Manage Projects Files 
- `Git Wrapper <./git-interface.html>`__
    - Print Git log
    - Commit changes
    - List tags 
    - Revert commits 
- Django Commands Wrapper
    - Start the Project 
    - Create Admin Account 
    - Check and execute migrations 
    - Format project code (using black) 
- Misc Helpers 
    - Generate random strings
    - Convert lists to string 
    - Tet current timestamp
