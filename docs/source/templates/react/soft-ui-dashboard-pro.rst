Getting Started
---------------

.. code-block:: shell

    # Obtain license and unzip the sources
    unzip soft-ui-dashboard-pro-react.zip

    # Navigate to project directory
    cd soft-ui-dashboard-pro-react

    # Install dependencies
    npm install

    # Start development server
    npm start

    # Build for production
    npm run build
