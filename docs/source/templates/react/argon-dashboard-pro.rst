Getting Started

.. code-block:: shell

    # Obtain license and unzip the sources
    unzip argon-dashboard-mui-pro.zip

    # Navigate to project directory
    cd argon-dashboard-mui-pro

    # Install dependencies
    npm install

    # Start development server
    npm start

    # Build for production
    npm run build


Project Structure
-----------------

.. code-block:: shell

    /src
        /assets           # Static assets (images, icons, fonts, CSS)
        /components       # Reusable UI components
        /contexts         # React context providers
        /examples         # Premium components with examples
        /layouts          # Multiple layout variations
        /routes           # Application routes with authentication
        /views            # Premium page components
        /data             # Sample data for demonstrations
        /services         # API services and integrations
        /utils            # Utility functions
        /hooks            # Custom React hooks
        /locales          # Internationalization files
        App.js            # Main application component
        index.js          # Application entry point
