Getting Started
---------------

.. code-block:: shell

    # Obtain license and unzip the sources
    unzip purity-ui-dashboard-pro.zip

    # Navigate to project directory
    cd purity-ui-dashboard-pro

    # Install dependencies
    npm install

    # Start development server
    npm start

    # Build for production
    npm run build
