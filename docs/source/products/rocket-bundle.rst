Rocket Bundle
--------------

Rocket is a powerful bundle of starters provided in modern technologies with production-ready modules. 

- `Django Rocket </docs/products/django/rocket/index.html>`__ - Free Django Starter styled with Tailwind/Flowbite that provides tools, charts and Docker Support
- `Django Rocket HTMX </docs/products/django/rocket/index.html>`__ - Free Django & HTMX sample
- `Django Rocket PRO </docs/products/django/rocket-pro/index.html>`__ - Production-ready Django Template styled with Tailwind/Flowbite
- `Flask Rocket </docs/products/flask/rocket/index.html>`__ - Open-Source Flask Starter styled with Tailwind/Flowbite
- `Flask Rocket PRO </docs/products/flask/rocket-pro/index.html>`__ - Premium Flask Starter enhanced with dynamic tables, OAuth, Docker and CI/Cd scripts for Render 
- `Rocket eCommerce </docs/products/rocket-ecommerce/index.html>`__ - Premium eCommerce starter integrated with Stripe
